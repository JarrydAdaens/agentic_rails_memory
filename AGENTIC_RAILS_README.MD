# Agentic Rails

This repository, `agentic-rails-memory`, is the **center rail** of Agentic Rails: the durable, git-revisioned store of cross-project artifacts (learnings and Resumes) that feeds data-driven routing.

The full framework overview — the three-rail model, the context tier system, provenance, and handover conventions — is canonical in `agentic_rails_tooling/AGENTIC_RAILS_README.MD`. Read it there; this stub is not duplicated content and will not be kept in sync independently.
